## 3. 联合曲线详解

联合曲线是 FairForge 的定价引擎，**完全公开、链上可验证**：

```
价格 = $0.0001 + (线性增长系数 × 已售出数量)
```

- 🟢 **起始价**：$0.0001/枚 — 最低入场价
- 📈 **线性上涨**：买的人越多，价格越高
- 🎯 **目标线**：储备金达到 **30 BNB** → 自动上 PancakeSwap！
- 💲 **DEX 交易税**：买入 1%，卖出 2%（用于回购销毁 FAIR）

**关键洞察**：联合曲线期间没有滑点税，这是你拿到最低价的唯一窗口。一旦上 DEX，每笔交易都要交税。

## 6. 借贷与做空

### 🏦 借贷（Long Pool）

- **出借方**：存入 BNB，赚取浮动利率
- **借入方**：抵押资产借入 BNB 做多
- 利率模型：利用率低时利率温和，利用率高时指数级上升
- 清算机制：抵押不足时自动清算，出借人资金安全

### 📉 做空（Short Pool）

- 从 Short Pool 借入代币
- 在 DEX 高价卖出
- 等价格下跌后低价买回
- 归还代币，差价就是利润

**风险提示**：做空亏损无上限，请严格设置止损！
