01 — Executive Summary

执行摘要

DogePad 是部署于 Arc 链上的代币发射平台，通过 Bonding Curve（联合曲线）实现公平发币，集成 DAO 治理投票、永续合约做空、动态资金费率与价格熔断机制，构建从资产发行到双向博弈的完整 DeFi 闭环。

核心命题

Pump.fun 在 Solana 上一年内创造了超过 $1.5 亿 协议收入，但其模式存在三大致命缺陷：无做空机制导致单边市场、无 DAO 筛选导致垃圾项目泛滥、无平台代币导致协议价值无法内化。DogePad 在此市场空白上构建下一代发射平台，以永续合约做空、DAO 治理和通缩飞轮三大创新填补市场空白。

04 — Technical Advantages

技术优势

DogePad 的技术架构围绕安全、公平、可控三大原则设计。所有合约基于 OpenZeppelin 标准库，集成 Timelock 治理、Transparent Proxy 升级模式和多重安全机制。

⚡

Dynamic Funding Rate

动态资金费率

基于多空持仓偏差自动调节资金费率。偏差越大，费率越高，促使市场回归平衡。偏差乘数 2x/1%，最大附加费率 0.5%/期，防止极端单边持仓。

🔥

Circuit Breaker

价格偏差熔断

当价格偏离 TWAP 预言机价格超过 15% 时自动触发熔断，暂停永续合约开仓。价格回归至 3% 偏差以内后自动恢复。有效防止闪崩攻击和剧烈操纵。

🛡

Insurance Fund

保险基金机制

每次清算产生的清算费用中，30% 自动注入保险基金。保险基金用于覆盖穿仓损失，保护盈利交易者不被亏损者拖累。现货交易手续费也按比例注入永续池。

🔒

Timelock Governance

时间锁治理

所有关键合约参数变更必须经过 Timelock 延迟执行，社区成员可在延迟期内审查和挑战提案。确保治理透明、不可突击修改，保护所有参与者利益。

🔄

Transparent Proxy

透明代理升级

采用 Transparent Proxy 模式实现合约可升级性。管理员和用户调用分离，防止选择器冲突。升级操作同样受 Timelock 约束，确保升级过程安全可控。

👥

Anti-Front-Running

反狙击设计

线性联合曲线 + 1% 买卖手续费 + 5% 持仓上限三重防护，有效抑制狙击机器人抢跑和大户操纵。所有代币从零开始，无预售、无团队预留。

风险披露

「All I want to know is where I'm going to die, so I'll never go there.」— 这不是一句玩笑话，而是投资的第一原则：先识别致命风险，然后远离它。以下是我们识别的每一个可能杀死项目的风险，以及我们正在采取的缓解措施。我们不回避风险，因为回避风险讨论本身就是最大的风险。

智能合约漏洞 HIGH 合约代码可能存在未发现的漏洞，导致资金损失。DeFi 历史上多次发生因合约漏洞导致的黑客攻击事件。 OpenZeppelin 标准库 + ReentrancyGuard + Pausable 紧急暂停 + Timelock 治理 + 计划第三方审计 + Transparent Proxy 可升级
