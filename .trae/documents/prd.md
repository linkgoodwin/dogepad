## 1. 产品概述

BSC Launchpad 是一个部署在 BSC 链上的代币发行与借贷一体化平台，采用改良版 Pump.fun 联合曲线模式实现公平发射，并内置做空/做多双池借贷协议 + 购入销毁引擎，形成"做空者成为做多燃料"的经济闭环。
- 解决问题：当前 Meme 币发行后必然归零（单向市场、卖压无对手），缺乏做空/做多博弈机制
- 核心理念：做错的人成为燃料为做对的人提供收益，做空成本→做多收益→购入销毁→通缩支撑
- 目标用户：BSC 链上的 DeFi 玩家、Meme 币创作者、寻求杠杆收益的交易者、做空博弈者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 代币创建者 | 连接钱包 | 创建代币、设置参数、领取创作者收益 |
| 做多者 | 连接钱包 | 买入代币、抵押新币借BNB、存入BNB赚利息 |
| 做空者 | 连接钱包 | 抵押BNB借出新币、卖出做空、承担指数利率 |

1. **首页**：平台数据概览、热门代币排行、最新上线代币、购入销毁实时数据

3. **代币交易页**：联合曲线交易界面、实时价格图表、交易历史
4. **借贷市场页**：做多池/做空池双池展示、利率曲线可视化、购入销毁统计
5. **借贷操作页**：做多（抵押新币借BNB）/做空（抵押BNB借新币）、健康因子监控
6. **持仓页**：用户代币持仓、做多/做空头寸、收益统计

### 3.3 指数利率算法

做空池日利率公式：
```
dailyRate = baseRate × e^(k × utilization²)

参数：
  baseRate = 0.01 (日化1%起步)
  k = 4.706
  utilization = 已借出量 / 可借总量

利率表：
  利用率 10% → 日利率 1.05%   (年化 3,798%)
  利用率 30% → 日利率 1.53%   (年化 5,620%)
  利用率 50% → 日利率 3.24%   (年化 12,194%)
  利用率 70% → 日利率 10.03%  (年化 41,045%)
  利用率 85% → 日利率 30.01%  (3天亏一半)
  利用率 95% → 日利率 69.80%  (1.5天亏完)
```

### 3.4 购入销毁引擎

资金来源：
- BNB 存款的 5%（每次存款自动触发）
- 做空利息的 30%（持续流入）
- 交易手续费的 20%

执行方式：
- 当累积 BNB > 阈值（0.1 BNB）时自动执行
- 调用 PancakeSwap 买入代币
- 将买入的代币转入黑洞地址销毁
- 链上事件可验证

反做空效果：
- 做空者卖出 → 购入销毁自动买入 → 直接对抗卖压
- 销毁减少供应 → 价格支撑
- 做空者付利息 → 30% 用来买币销毁 → 做空者资助对抗自己的力量

### 3.5 改良方案核心机制

**相比 Pump.fun 的改良点：**

1. **防狙击机制**：开盘前 N 个区块限制单笔买入量，设置交易冷却时间
2. **创作者激励与锁仓**：创作者获得一定比例代币，但按线性解锁，防止砸盘
3. **资金分流模型**：内盘打满后分流至外盘LP + 做多池 + 做空池，而非全部进LP
4. **做空/做多双池**：做空池可借新币（指数利率），做多池可借BNB（低利率）
5. **指数利率曲线**：做空利率随利用率指数级暴涨，大量做空=自焚
6. **购入销毁引擎**：BNB存款5% + 做空利息30% → 自动买币销毁 → 通缩支撑
7. **空头挤压**：币价上涨→做空者被清算→强制买回→推高币价→做多火箭燃料
8. **动态 LTV**：做多池抵押率根据流动性深度、波动率、上线时间动态调整

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景 (#0a0a0f) + 霓虹绿 (#00ff88) 强调色 + 紫色 (#8b5cf6) 辅助色 + 红色 (#ff4444) 做空色
- **按钮风格**：圆角按钮，做多用霓虹绿，做空用红色，hover 时增强发光
- **字体**：标题使用 Space Grotesk，正文使用 DM Sans
- **布局风格**：左侧导航栏 + 主内容区，卡片式布局
- **图标风格**：线性图标 (Lucide)，搭配霓虹发光效果
- **动效**：页面切换淡入，购入销毁火焰动画，利率曲线动态绘制
